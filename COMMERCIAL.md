# Commercial Use

RustShield Gateway public repository is a technical preview for controlled
SITL/laboratory validation.

The public repository is intentionally scoped. Commercial work happens through
explicit assessment, laboratory pilot or partner-integration engagements.

## Available Engagements

### MAVLink Security Assessment

Remote-first review of a MAVLink-based architecture, including threat model,
critical/high-risk command review, signing posture, SITL/lab evidence review
and remediation planning.

Typical deliverables:

- MAVLink threat model;
- architecture review;
- critical/high-risk command matrix;
- signing posture review;
- controlled SITL/lab test plan;
- evidence pack;
- final technical report.

### RustShield Lab Pilot

Controlled integration of RustShield in a SITL/laboratory workflow with policy
tuning, signing audit/enforce validation, shadow-enforce evaluation, read-only
observability and evidence capture.

Typical deliverables:

- SITL/lab integration;
- policy tuning;
- signing audit/enforce validation;
- shadow enforcement report;
- read-only observability setup;
- evidence pack;
- technical handover session.

### Partner Integration

Collaboration with UAV integrators, cybersecurity labs, drone C2 platforms or
industrial security teams evaluating RustShield as a security-assessment or
lab-validation module.

Possible models:

- paid pilot;
- private integration;
- internal lab license;
- revenue-share discussion;
- later licensing discussion.

## Not Offered Without Separate Agreement

- Real UAV operation.
- Flight safety validation.
- Certification.
- Hardware/radio testing.
- Production deployment responsibility.
- Exclusive licensing.
