# Responsible Use

RustShield Gateway is security-sensitive and dual-use. Use it only for
authorized documentation, simulation, laboratory validation and defensive
security review.

## Do Not Use This Repository For

- operating real UAV hardware;
- arming, takeoff, landing, mission upload, mode changes, parameter mutation or
  RC override against real vehicles;
- bypassing security controls;
- offensive testing without authorization;
- publishing secrets, payloads, signing keys or customer data;
- claiming flight safety or certification.

## Expected Use

- controlled SITL/laboratory validation;
- architecture and threat-model review;
- policy evaluation;
- signing posture assessment;
- read-only evidence capture;
- partner or customer pilots under explicit scope.

Users are responsible for compliance, safety procedures, export-control review
and authorization in their own jurisdiction and environment.
