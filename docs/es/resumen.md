# Resumen Público

RustShield Gateway es una vista técnica pública de un gateway de seguridad
MAVLink escrito en Rust para validación controlada en SITL/laboratorio.

## Alcance

- revisión técnica;
- simulación y laboratorio controlado;
- políticas semánticas para comandos MAVLink críticos seleccionados;
- observabilidad read-only;
- evidencias públicas resumidas.

## No Incluye

- operación de UAV real;
- certificación;
- validación hardware/radio;
- cobertura completa PX4;
- cobertura completa MAVLink;
- garantía de tiempo real extremo a extremo.
