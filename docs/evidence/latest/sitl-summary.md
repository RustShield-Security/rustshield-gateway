# SITL Summary

RustShield Gateway has a public-preview scope centered on controlled
SITL/laboratory validation.

Publicly claimable scope:

- UDP/SITL gateway workflow;
- ArduPilot Copter SITL as primary documented target;
- QGroundControl-oriented laboratory topology;
- selected critical/high-risk command policy behavior;
- read-only observability for evidence capture.

Not claimable from this public evidence:

- real UAV validation;
- flight readiness;
- hardware/radio validation;
- complete autopilot coverage.
