# Signing Summary

RustShield Gateway is signing-aware in laboratory modes:

- observe;
- audit;
- enforce for selected critical/high-risk GCS-to-vehicle commands.

Allowed public claim:

```text
MAVLink signing observability and laboratory enforcement paths are implemented
for controlled validation.
```

Not allowed:

- operational key-management completeness;
- production authentication readiness;
- hardware-validated signing enforcement;
- full MAVLink signing ecosystem coverage.
